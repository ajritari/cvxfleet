# Convex formulation of the maritime fleet size and mix problem considering battery electric ships

This repository contains the source code for the experiments in our paper: 

**Convex formulation of the maritime fleet size and mix problem considering battery electric ships** <br>
*Antti Ritari, Jani Romanoff, Kari Tammi* <br>
Paper: [https://doi.org/10.1016/j.oceaneng.2026.124415](https://doi.org/10.1016/j.oceaneng.2026.124415) <br>

## Repository Overview

We provide <code>cvxfleet_baltic_ferry_service.ipynb</code> for reproducing the Baltic Sea ferry service design case study results, and <code>cvxfleet_linear_benchmark.ipynb</code> for reproducing the solve time and fleet cost benchmarking results.
